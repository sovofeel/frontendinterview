title: Как проверять значения на равенство null и undefined?
complexity: junior
category: js
lang: ru
type: question
tags: ['falsy values']
---

Выражение в скобках будет приведено к true в том случае, если оно не является чем-то из следующего списка:

- null
- undefined
- NaN
- Пустая строка
- 0
- false
