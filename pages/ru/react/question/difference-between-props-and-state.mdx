title: Что такое props и state в React
complexity: junior
category: react
lang: ru
type: question
tags: ['react state', 'react state']
---

- props — это объект, передающийся от родительского компонента к дочернему, у дочернего компонента нет доступа его изменению.
- state - Это объект состояния компонента, изменение которого вызывают ререндер компонента.
