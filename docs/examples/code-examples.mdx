```js example
  // В какой последовательности выведутся символы
  console.log("A");
  setTimeout(() => console.log("B"), 0);
  setTimeout(() => console.log("C"), 0);
  console.log("D");)
```

```js answer
'A', 'D', 'B', 'C'
```
